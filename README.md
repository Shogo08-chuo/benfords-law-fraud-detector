# Benford's Law Fraud Detector
Gemini APIによるAI解析を統合した、統計的データ監査シミュレーター

## 概要
本プロジェクトは、財務データや統計データの「第1桁」の出現頻度が自然法則（ベンフォードの法則）に従っているかを検証するWebアプリケーションです。
単なる統計解析にとどまらず、**Gemini API (Google AI)** を活用することで、検知された異常値に対して「どのような不正のリスクがあるか」という具体的な解説を自動生成し、データ監査の効率化を支援します。

## 主な機能
- **ベンフォードの法則による解析**: アップロードされたCSVデータの第1桁の分布を、理論値と比較・可視化します。
- **統計的乖離の定量化**: データが理論値からどれだけ「統計的に不自然か」を定量的に算出します。
- **AIによる不正シナリオ解説**: Gemini APIを活用し、統計データに基づいた具体的な不正リスクや監査上の注意点をAIがアドバイスします。

## 技術スタック
- **Language**: Python
- **Framework**: Streamlit
- **AI**: Gemini API (google-generativeai)
- **Library**: Pandas, NumPy, SciPy, Matplotlib
